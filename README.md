# kualp-asc-project

> 근로자의 현재 근로 환경 및 지위를 진단하는 MBTI형 설문 테스트

본 프로젝트는 고려대학교 법정책학회 **KUALP**와 **ASC**가 협업하여 진행하는 정책 제안용 웹사이트입니다.
'근로 시간', '물품 제공 주체' 등 다양한 문답을 통해 근로자가 현재 자신의 법적/환경적 위치를 확인하고, 총점을 도출하는 것을 목표로 합니다.

## 🧑‍🤝‍🧑 협업

* **KUALP (고려대학교 법정책학회):** 기획, 콘텐츠(질문지), 정책 이론 연구
* **ASC (아스키):** 웹사이트 개발, 배포 및 유지보수
